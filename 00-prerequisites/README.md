# Prerequisites

## What You Need Before the Workshop

### Access Requirements

- [ ] A Microsoft 365 account with Power Platform access
- [ ] Access to the **shared developer environment** (provided by your facilitator)
- [ ] A modern web browser (Microsoft Edge or Google Chrome recommended)

### Accounts & Licensing

Your facilitator will provide:
- The **environment URL** for the shared developer environment
- Your **login credentials** (if using training accounts)
- The **environment name** to select in Power Platform

### Browser Setup

1. Open [make.powerapps.com](https://make.powerapps.com) and sign in
2. Open [make.powerautomate.com](https://make.powerautomate.com) and sign in
3. Verify you can see the shared developer environment in the environment picker (top-right)

### What the Facilitator Needs to Prepare

- [ ] A Power Platform developer environment (or trial environment)
- [ ] User accounts with **System Customizer** or **Environment Maker** security role
- [ ] Confirm all attendees can access the environment before the workshop
- [ ] Pre-assign attendee prefixes (see [Environment Setup](../02-lab-environment-setup/README.md))

### Recommended (Not Required)

- Basic understanding of databases (tables, columns, relationships)
- Familiarity with business workflows and approval processes
- Experience with any low-code or no-code tools (helpful but not necessary)
