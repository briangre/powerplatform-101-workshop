# Completed Solution

This folder is reserved for the fully completed workshop solution export, including all components:

- Dataverse tables with sample data schema
- Model-Driven App with custom forms and views
- Canvas App with 3 screens (Home, Request Form, Confirmation)
- Power Automate cloud flow (notification + status update)

## How to Create the Completed Solution

1. Complete all labs (1-5) in the workshop environment
2. Go to **Solutions** in the maker portal
3. Select your solution
4. Click **Export** → **Unmanaged**
5. Save the `.zip` file to this folder

## Use Cases

- **Facilitator reference:** Verify your instructions produce the expected result
- **Catch-up:** Import for attendees who fall significantly behind
- **Post-workshop review:** Attendees can import and explore at their own pace

> **Note:** Solution exports may contain environment-specific connection references that need updating after import.
