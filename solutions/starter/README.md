# Starter Solution

This folder is reserved for a pre-built Dataverse solution export that can be used as a starting point if attendees fall behind or if the facilitator wants to skip the Dataverse setup lab.

## How to Create the Starter Solution

1. Complete Labs 1 (Dataverse Tables) in the workshop environment
2. Go to **Solutions** in the maker portal
3. Select your solution
4. Click **Export** → **Unmanaged**
5. Save the `.zip` file to this folder

## How to Import

1. Go to [make.powerapps.com](https://make.powerapps.com)
2. Select the target environment
3. Click **Solutions** → **Import**
4. Upload the `.zip` file
5. Follow the import wizard

> **Note:** Attendees will need to update publisher prefixes and rename components to match their assigned prefix after importing.
